## 功能特性

- **全自动评教** — 自动填写所有单选题（默认选最高分），支持自动提交或手动确认
- **自定义题目选项** — 每道题可单独配置选第几个选项，通过设置弹窗下拉框直观调整
- **速度可调** — 提交倒计时 1~10 秒可调，支持快速模式跳过等待
- **主观题自定义** — 可自定义"意见或建议"文本内容
- **跨域自动续接** — 列表页（`jwc.cuit.edu.cn`）和评教页（`jwgl.cuit.edu.cn`）双域名自动衔接，无需手动操作
- **进度持久化** — 评教进度跨页面保存，刷新或跳转不丢失
- **进度可视化** — 悬浮球实时显示完成进度，含 SVG 环形进度条
- **页面检测** — 自动识别评教页面与列表页面，提示清晰
- **结构检测工具** — 内置页面结构检测，方便调试和排查问题
- **错误日志** — 操作失败自动记录，点击可查看详情
- **庆祝动画** — 全部完成后播放彩色纸屑动画

## 技术架构

```
列表页 (jwc.cuit.edu.cn)          评教页 (jwgl.cuit.edu.cn)
┌─────────────────────┐          ┌─────────────────────┐
│  table.gridtable    │  点击链接  │  #question-list     │
│  评教链接列表        │ ───────→ │  Backbone.js 渲染   │
│                     │          │  Radio + Textarea   │
│  悬浮球控制面板      │ ←─────── │  #sub 提交按钮      │
│  自动继续下一位      │  页面跳转  │  自动填写 + 提交    │
└─────────────────────┘          └─────────────────────┘
         ↕ GM_setValue 持久化状态
    autoSubmit / isProcessing / completedCount / questionSelectors
```

### 核心适配点

| 功能 | 正方教务系统（CUIT）选择器 |
|------|---------------------------|
| 列表页检测 | `table.gridtable` + `a[href*="stdEvaluate!answer"]` |
| 评教页检测 | `#app-main` + URL 含 `stdEvaluate!answer` |
| 单选题 | `#question-list input[type="radio"]` 按 name 分组 |
| 主观题 | `#question-list textarea` |
| 提交按钮 | `#sub` |
| 题目元数据 | `QUESTIONS` 全局变量（Backbone Collection） |

## 安装

### 环境要求

- 浏览器安装了 [Tampermonkey](https://www.tampermonkey.net/) 扩展

### 安装步骤

1. 点击 [`CUIT-auto-PJ.user.js`](CUIT-auto-PJ.user.js) 查看源码
2. 复制全部内容
3. 打开 Tampermonkey 控制台 → **新建脚本** → 粘贴 → **保存**
4. 访问教务评教页面，脚本自动运行

## 使用方法

### 基本流程

1. 打开教务系统，进入待评教老师列表页面
   - 地址：`https://jwc.cuit.edu.cn/eams/quality/stdEvaluate.action`
2. 点击右下角**悬浮球**展开控制面板
3. **开启「自动提交」开关**（重要！否则只填写不提交）
4. 点击**「开始评教」**，脚本自动完成所有评教

### 自定义题目选项

1. 在控制面板点击**设置按钮**（齿轮图标）
2. 顶部显示所有题目的下拉选择框
3. 每道题可单独选择第几个选项
4. 点击**「保存设置」**

默认配置：

| 题目 | engName | 默认选项 |
|------|---------|----------|
| 批改作业次数 | sy13 | 第3个（2-3次） |
| 每周课外学习时间 | sy14 | 第2个（1小时以内） |
| 需要非常努力才能达到课程要求 | sy16 | 第3个（一般） |
| 其他题目 | — | 第1个（最高分） |

### 快速模式

在设置中开启**「快速模式」**，跳过倒计时直接提交，速度最快。

### 调试工具

**双击设置按钮**（齿轮图标）可打开测试后门：

- 🔔 测试通知

- 📋 查看错误日志
- 🔍 **检查页面结构** — 显示当前页面的 DOM 信息，用于排查问题
